# FGS ONE

Eine Next.js-Anwendung für FGS ONE.

## Entwicklung

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Deployment

```bash
npm run build
npm start
```

## Struktur

- `/app` - Next.js App Router Seiten
- `/public/assets` - Statische Assets (Favicons, Icons)
- `/app/globals.css` - Globale Styles
